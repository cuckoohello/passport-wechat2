# passport-wechat

A Fork of [passport-wechat](https://github.com/liangyali/passport-wechat), but compatible with 
the [loopback](loopback.io) framework.

## Installation

```sh
$ npm install passport-wechat2
```

## Usage 

```js
var WechatStrategy = require('passport-wechat');
```
